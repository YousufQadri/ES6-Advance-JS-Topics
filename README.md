- Function & Variable Hoisting
- Arrow Function
- Default Values of a Parameter
- Ternary Operator
- Object Literal Shorthand
- Destructuring Array & Object
- Template Literals
- Try / Catch / Throw
- Rest Parameters
- Spread Operator
- Short Circuit Evaluation
- For In & For Of Loop
- Promises
